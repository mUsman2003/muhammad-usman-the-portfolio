
## Muhammad Usman — Editorial Portfolio

### Design Foundation
- Dark theme (`#0a0a0a` bg, `#00f0ff` cyan, `#b8ff57` lime accents)
- Google Fonts: **Bebas Neue** (display) + **Inter** (body)
- Custom CSS variables, smooth scroll, grain texture overlay

### Pages & Components

**1. Intro Loader**
- Fullscreen black overlay with "MUHAMMAD USMAN" animating in letter by letter
- Splits apart vertically to reveal the page underneath

**2. Custom Cursor**
- Small dot + trailing circle, expands/changes color on hoverable elements
- Disabled on touch devices

**3. Navbar**
- Transparent → frosted glass blur on scroll
- `MU` cyan monogram logo
- Mobile: fullscreen overlay with giant nav links

**4. Hero Section**
- Giant stacked Bebas Neue headlines: `AI ENGINEER. / BUILDER. / PROBLEM SOLVER.`
- Subtext, two outlined CTA buttons with fill-on-hover
- Floating cyan particle canvas background
- Bottom marquee: tech logos scrolling endlessly

**5. About Section**
- Oversized muted `ABOUT` label
- Two-column: bold statement left, bio right
- 4 animated counting stat cards: `20+ Projects`, `5★ Fiverr`, `2 AWS Certs`, `3+ Years`

**6. Services Section**
- `WHAT I DO` label
- 3 horizontal expandable cards with bold `01/02/03` numbers
- Hover expands to reveal tools list

**7. Projects Section**
- `SELECTED WORK` label
- Magazine staggered layout: 1 full-width featured card + grid below
- 3D tilt + cyan background shift on hover
- Tech tag pills, arrow link

**8. Skills Strip**
- Two infinite marquee rows scrolling in opposite directions
- AI/ML row left, DevOps/Cloud row right

**9. Certifications**
- Dark cards with glowing cyan border animation
- AWS x2 + Fiverr Level 1 badge

**10. Contact Section**
- Giant `LET'S BUILD SOMETHING.` heading
- Large email link with draw-on underline hover
- 3-field dark form (Name, Email, Message) with cyan focus glow
- GitHub + LinkedIn icon buttons

**11. Footer**
- Minimal 3-column: copyright, location, social links
- Grain overlay texture

### Scroll Animations
- Intersection Observer on every section for fade+slide-up reveals
- Parallax on hero background elements
- Count-up on stat numbers when in view
